Developed a COVID-19 Impact Analysis Dashboard using Python's Dash and Plotly libraries, offering real-time statistics and 
interactive visualizations to analyze the pandemic's impact.
  
Implemented features like total cases, active cases, recoveries, and deaths, with dropdown menus for data exploration.

•	Environment:- Pycharm, Jupyter Notebook

•	Pyhton Libraries:- Numpy, Pandas, Plotly, Dash

•	Machine Learning Algorithms:- text-based preprocessing

•	Pre-processing steps:- Stemming & lemmatization, Tokenization, Stop words
